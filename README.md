# AI Resume Builder (MERN Stack)

An AI-powered Resume Builder application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). This project helps users create, customize, enhance, and share professional resumes with the assistance of AI. It offers multiple templates, real-time preview, profile image processing, PDF export, and secure user authentication.

## 🚀 Features

### 🔐 User Authentication

* Secure user registration and login system.
* Individual account management and protected routes.
* JWT-based authentication for enhanced security.

### 📝 Resume Creation & Management

* Create resumes from scratch.
* Upload existing resume PDFs and automatically extract information.
* Add, edit, update, and delete resumes from the dashboard.
* Manage multiple resumes under a single account.

### 🤖 AI-Powered Resume Enhancement

* Integrated with Google Gemini AI.
* Generate and improve professional summaries.
* Optimize job descriptions and work experience content.
* Enhance resume content for better ATS (Applicant Tracking System) compatibility.

### 🖼️ Profile Image Processing

* Image storage and management using ImageKit.
* Automated background removal.
* Face-centered image optimization for professional profile photos.

### 🎨 Resume Customization

* Multiple professionally designed resume templates.
* Dynamic color themes and styling options.
* Personalized layouts to match different career profiles.

### 👀 Live Preview

* Real-time resume preview while editing.
* Instant reflection of content and design changes.

### 🌐 Sharing & Accessibility

* Public/Private resume visibility settings.
* Generate unique shareable URLs for public resumes.
* Easy online access to resumes through live links.

### 📄 PDF Export

* Download resumes as PDF files.
* Print-ready resume formatting.
